# Google SWE Prep Roadmap

A comprehensive, interactive, and responsive roadmap designed to guide you through your Google Software Engineer (SWE) interview preparation.

## Features

-   **Interactive Checklist**: Track your progress through months of curated topics, from basic C++ and DSA to System Design and Mock Interviews.
-   **Dynamic Scheduling**: 
    -   Set your customized **Start Date**.
    -   All weeks and "dates" in the roadmap automatically adjust to your schedule.
    -   Handles custom durations (e.g., 7-day weeks, 10-day sprints).
-   **Mobile Friendly**: Fully responsive design that looks great on desktop and mobile.
-   **Progress Tracking**:
    -   Visual progress bars for each month and the overall roadmap.
    -   **GitHub Sync**: Save your progress across devices by syncing to a private GitHub repository (`progress.json`).
-   **Motivation**:
    -   Daily quote/routine reminders.
    -   **Dynamic Footer Countdown**: Always know exactly how many days are left until your goal completion date.
